CleverTech Dojo - Site Plan

Site Name

CleverTech Dojo

Reasoning: The name combines "CleverTech," suggesting intelligence and technology focus, with "Dojo," a place for immersive learning and practice. This reflects the site's purpose as a dedicated space for tracking progress while learning programming skills.

Site Purpose

CleverTech Dojo aims to be a dynamic online platform designed to help individuals track their learning progress in specific technology fields, particularly Frontend Development (HTML, CSS, JavaScript), Python, and C# (OOP basics).

The site will provide structured learning paths ("roadmaps") for these technologies, allowing users to visually track their advancement via progress bars as they mark topics complete. It will also offer the flexibility for users to create and monitor custom learning paths tailored to their unique goals. The core purpose is to combat the overwhelming feeling often associated with learning programming by offering a clear, step-by-step tracking system, thereby keeping users focused and motivated. Ultimately, it serves as both a tool for the community and a personal project to reinforce web development skills.

Scenarios

These scenarios represent questions potential users might have:

  • "I want to learn Python but don't know where to start. Does this site offer a beginner's path for Python, and how can I see the topics involved?"
  • "I'm learning React, which isn't listed as a main path. Can I create my own custom roadmap on CleverTech Dojo to track my React learning milestones?"
  • "How can I visually see how much of the C# OOP path I have completed?"

Color Schema

  • Primary Color: #C84B8C (Pink/Purple)
    • Usage: Headings (h1, h2, h3), Buttons, Important Accents, Section Borders, Footer/Header Background.
  • Secondary Color: #6A7FDB (Blue)
    • Usage: Progress Bars, Secondary Accents, Heading Underlines.
  • Accent Color: #F4F4F4 (Light Gray)
    • Usage: Background for content sections to differentiate from the main background.
  • Text Color: #333333 (Dark Gray/Near Black)
    • Usage: Main body text, paragraphs, list items.
  • Background Color: #FFFFFF (White)
    • Usage: Main page background.

Typography

  • Headings (h1, h2, h3, etc.):
    • Font: 'Poppins', sans-serif (from Google Fonts)
    • Usage: Main page titles, section headings.
  • Body Text (paragraphs, lists, etc.):
    • Font: 'Roboto', sans-serif (from Google Fonts)
    • Usage: All standard paragraph text, descriptions, list items.
