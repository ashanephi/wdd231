CleverTech Dojo - Site Plan

Discover the structure and features of CleverTech Dojo.

Site Name

CleverTech Dojo

Reasoning: The name combines "CleverTech," representing intelligence and technology, with "Dojo," a place for immersive learning and practice. This reflects the site's purpose as a dedicated space for tracking progress while learning programming skills.

Site Purpose

CleverTech Dojo is an online platform designed to help individuals track their learning progress in technology fields such as Frontend Development, Python, and C#. The site provides structured roadmaps for these technologies, allowing users to visually track their progress using progress bars. Additionally, users can create custom roadmaps tailored to their unique learning goals.

The purpose is to simplify the learning process by offering clear, step-by-step guidance, reducing the feeling of being overwhelmed, and keeping users motivated.

Site Features

  • Predefined Roadmaps: Structured learning paths for Python, C#, and Frontend Development.
  • Custom Roadmaps: Users can create and save personalized roadmaps to suit their learning goals.
  • Progress Tracking: Visual progress bars to track completion of topics and sections.
  • Resource Library: Curated resources for frontend, backend, and general programming.
  • Dark Mode: Toggle between light and dark themes for better accessibility.
  • Mobile Responsiveness: Fully responsive design for seamless use on any device.

User Scenarios

  • "I want to learn Python but don't know where to start. Does this site offer a beginner's roadmap for Python?"
  • "Can I create a custom roadmap to track my learning progress for React?"
  • "How can I see how much of the Frontend Development roadmap I have completed?"
  • "Are there resources available for learning backend development with Node.js?"

Color Schema

  • Primary Color: #C84B8C (Pink/Purple) - Used for headings, buttons, and accents.
  • Secondary Color: #6A7FDB (Blue) - Used for progress bars and secondary accents.
  • Accent Color: #F4F4F4 (Light Gray) - Used for background sections.
  • Text Color: #333333 (Dark Gray) - Used for main text.
  • Background Color: #FFFFFF (White) - Used for the main page background.

Typography

  • Headings: 'Poppins', sans-serif - Used for titles and section headings.
  • Body Text: 'Roboto', sans-serif - Used for paragraphs and list items.
